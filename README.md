# No Text Highlight Color Chrome Extension

Removes text highlight color from all websites, making selected text background transparent.

## Features

- Removes default highlight color on text selection.
- Forces `user-select: text` on all elements.

## Installation

1. Clone or download this repository.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable "Developer mode" (top right).
4. Click "Load unpacked" and select this folder.
